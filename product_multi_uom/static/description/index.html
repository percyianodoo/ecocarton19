Configuration

Enable 'Unit of Measure' In Configuration Settings of Sale.

Enable the Need Secondary UoM's

Enable the 'Need Secondary UoM's' Boolean Field in the Product Form View.

Base UoM's of the Product

We can see the Base UoM's of the Product and it's Conversion.

Add Products.

By Clicking the 'Add a line' we have a Popup for adding the Secondary Unit of Measures.

Set the Values for the Secondary UoM

Set the Values for the Secondary UoM, and it's Ratio based on the Base Unit.

Secondary UoM

Like this we can add more Secondary UoM for this Product.

Sale Order

When Creating a Sale Order, you'll notice Fields for 'Secondary UoM' and 'Secondary Quantity' in the Order Line. Alongside, the Base UoM and its Corresponding Quantity are also Displayed for Reference. Add any Product, Secondary UoM and Secondary Quantity.

Frequently Asked Questions

What does this module allow to do?

This module enables you to sell your products based on weight (kilograms), length (meters), or as individual units, providing flexibility in how you price and sell your items.

Can I choose more than one selling method for my products?

Yes, you can offer different selling methods for each product, allowing customers to choose how they want to purchase based on their needs.
